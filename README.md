## Usage

``` bash
# install dependencies
npm install

# serve with vite hmr
npm run dev

# serve with vite preview
npm run preview

# build:dev - HTML/CSS/JS is NOT minified 
npm run build:dev

# build - HTML/CSS/JS is minified
npm run build

```
